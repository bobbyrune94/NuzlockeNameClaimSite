Want Your Viewers and Community to Nickname Your Playthrough Team?

If you bring this discord bot into your server, you can let your community use commands to claim and nickname any Pokemon. People can make claims, view claims, edit their nicknames, change their claims, and remove claims through the bot using structured Discord commands.

Selecting a command will automatically bring up all the parameters to populate.

Tech Breakdown

System Diagram

The PokeNameBot System is fairly simple. Broken down to its simplest components, it’s a database that stores data and a discord bot that views and modifies that data. Everything else around it is used to help with that functionality.

  1. The Website: This is what you’re looking at right now. The website was built to give users an easy way to view and search for existing and available claims.
  2. The Discord Bot: This is the primary interface for the system. Through the discord bot, you can make claims, edit existing claims, and view claim data for yourself and any Pokemon available.
  3. API Gateway: This is how public systems, such as the website and discord bot, can interface with the Databases. Not everyone should be able to manipulate the data, so this is our way of restricting access to it. API endpoints call Lambda Functions (described below) to perform any number of functions on database tables.
  4. Lambda Functions: This is how we actually manipulate and grab data from the databases. Depending on the functionality, the lambdas can edit tables to make/change a claim or get single entries or whole tables. I’ve also made lambdas to make onboarding new Pokemon and discord servers easier for my future sanity.
  5. Databases: This is where all the data is stored. Think of it like an excel spreadsheet that I can programmatically edit and get data from.

Evolution Lines Table

This table stores all the evolution lines and essential information for each pokemon. The most important data here is the evo-line field. Any time a claim comes in, it checks this table to see what other Pokemon are in that evolutionary line that will also get claimed.

Claims Tables

These tables store every single pokemon available to claim. Unclaimed pokemon will have UNDEFINED in the nickname while claimed pokemon (see Jolteon) will display the nickname and claimer’s discord username and ID. It also stores the next time the user can change their claim, as changing claims is restricted to once every 3 months.

This website takes the data from these tables and displays it in a more easily consumable and searchable format.
